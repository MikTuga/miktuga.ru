Tuga Store

v0.2.1 · central

Главное приложение экосистемы — каталог + диагностика устройства + форма обратной связи. Ставится первым.

  • Каталог с поиском, фильтрами, сортировкой
  • Диагностика: 10+ секций (CPU, дисплей, USB, root, сенсоры…)
  • Установка APK через PackageInstaller.Session
  • Feedback с offline /sdcard fallback

Tuga Settings

v0.1.1 · central

Общие настройки экосистемы (единицы измерения, пути) + ContentProvider. Ставится сразу после TugaStore.

  • Inline segmented controls (никаких dialogs)
  • Folder picker для USB / Music / Reports
  • ContentProvider permission=normal — открыт community apps
  • Типизированная схема в tuga-design

Tuga OBD

v0.1.0 · community

OBD-II дашборд: скорость, обороты, температура, топливо. Сейчас demo-режим, ELM327 Bluetooth — в следующей версии.

  • 2×2 metric grid с большими цифрами
  • DTC bar + voltage display
  • Sin-wave demo data (для тестов)
  • Читает units из Tuga Settings (км/ч ↔ mph)

Tuga GPS

v0.1.0 · community

Координаты, скорость, высота, точность, спутники + компас с magnetic declination correction.

  • TYPE_ROTATION_VECTOR компас

Tuga Media

v0.1.0 · community

Сканер музыки и видео на USB-флешке. Передаёт в стоковый плеер головы через ACTION_VIEW.

  • mp3/m4a/flac/ogg/wav + mp4/mkv/avi/webm/mov
  • Filter chips: All / Audio / Video
  • Fallback на /sdcard/Music если USB нет
  • Тап → системный плеер

Tuga Sync

v0.1.0 · community

Bidirectional sync файлов USB ↔ устройство. Два preset профиля: Music и Reports.

  • Skip-if-same-size optimization
  • Progress по file count
  • Truncation warning (no silent data loss)
  • Пути из Tuga Settings

Установка на голову

  1. 1. Подготовить USB-флешку

    Формат FAT32 (ExFAT не читается). Объём ≤ 32 ГБ — некоторые головы не подхватывают больше.

  2. 2. Скачать APK на флешку

    Скачать APK из раздела выше (или со страниц Releases каждого репо). Структура на флешке:

    <корень флешки>
├── TugaStore.apk            ← ставится первым
└── apps/
    ├── tugasettings.apk     ← вторым (создаёт ContentProvider)
    ├── tugaobd.apk
    ├── tugagps.apk
    ├── tugamedia.apk
    └── tugasync.apk

  3. 3. Установить TugaStore

    Воткнуть флешку в голову → открыть ApkInstaller / Root Explorer → тапнуть TugaStore.apk → Установить.

  4. 4. Через Tuga Store установить остальные

    Открыть Tuga Store → Каталог приложений → нажать УСТАНОВИТЬ для каждой утилиты. Tuga Settings первым.

  5. 5. (опционально) Обновления по WiFi

    В версии v0.4+ Tuga Store сам проверяет miktuga.ru/api/manifest.json и предлагает обновиться без USB. Сейчас — только USB.

Зачем USB первый раз? Голова не имеет Google Play, поэтому первый раз нужен sideload. После установки Tuga Store будет ставить остальные апки сам.
